Using NYC Building Data to Predict Multipath GPS Error in NYC Taxi Data (May 2015, with E. Bruzelius, S. Grandhi, D. Sheehan)

We studied multipath GPS errors in NYC Taxi GPS location data with the understanding that height of buildings increase error in GPS reads. We leveraged Leaflet, Leaflet DVF, OGR-GDAL, Google BigQuery and Shiny.
